# Honeydrop Learning House

A learning centre in Belleville, Ontario offering homeschool support, after-school enrichment, adult learning, tutoring, and summer camps.

## Project Structure

- `frontend/` — Next.js application with pages, enrolment forms, and Square payment integration.

## Getting Started

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Environment Variables

See `frontend/README.md` for the full list of Square payment and notification environment variables.
